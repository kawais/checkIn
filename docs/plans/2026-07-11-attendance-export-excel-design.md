# 考勤明细查询结果导出 Excel 设计文档

## 背景

为了方便班级管理员/教师对学生的考勤数据进行整理与归档，需要在考勤明细查询页面增加导出功能。在查询出考勤结果后，提供一个导出按钮，支持将查询区间内所有学生的出勤统计与每日考勤明细（包含备注）以 Excel 表格形式下载。

同时，为了优化查询页面的交互体验，需要调整页面布局，使页面整体高度固定，仅考勤报表数据区域可滚动。

---

## 需求规格

### 1. 滚动区域限制
*   查询页面的外层容器高度固定为 `100vh` / `100dvh`，并隐藏全局滚动条。
*   顶部 Header、班级概要信息、日期过滤条件面板固定在页面上方。
*   下方的考勤明细报表容器（`.report-panel`）占满剩余高度，内部表格区域（`.table-container`）允许纵向和横向滚动。

### 2. 导出按钮展示与触发
*   **文案**：`导出 Excel`。
*   **显示条件**：当考勤数据查询完成且学生列表不为空时（`!isLoading && studentsData.length > 0`），在报表区域右上方显示该按钮。
*   **视觉风格**：采用符合现有系统的微晶玻璃质感（Glassmorphism）按钮，并配有下载图标。

### 3. 导出 Excel 数据格式
导出生成的 Excel 格式严格对应需求设计的矩阵式结构：
*   **列布局**：A列为日期与汇总项目；B列及以后为学生列，每一列代表一位学生。
*   **第 1 行 (Row 1)**：
    *   `A1` 单元格为空。
    *   `B1, C1, D1...` 填充学生姓名。
*   **第 2 行 (Row 2)**：
    *   `A2` 为 `"签到天数"`。
    *   `B2, C2, D2...` 填充每个学生在查询区间内已签到的累计天数（即 status 为 `true` 的记录数）。
*   **第 3 行 (Row 3)**：
    *   `A3` 为 `"未签到天数"`。
    *   `B3, C3, D3...` 填充每个学生在**有开启过签到的日期**中未签到的累计天数（有签到记录但 status 为 `false` 或缺失数据）。
*   **第 4 行及以下 (Row 4+)**：
    *   **A 列 (A4 起)**：显示日期，格式严格为 `"M月D日"`（如 `1月1日`、`7月11日`），包含查询范围内的**所有自然日**。
    *   **学生单元格 (B4 起)**：
        *   若当天**未开启签到**（所有学生均无考勤记录）：行内所有学生的单元格均**保持空白**。
        *   若当天**开启过签到**（至少有一个学生有考勤记录）：
            *   若该学生在当天状态为**已签到**：显示 `✓ 备注内容`（若备注为空则仅显示 `✓`）。
            *   若该学生在当天状态为**未签到**：显示 `✗ 备注内容`（若备注为空则仅显示 `✗`）。
            *   若该学生在当天没有记录（例如中途进班）：显示 `✗`。

## 技术实现方案

本系统采用纯前端导出方案，利用依赖项中的 `xlsx` (SheetJS) 库直接在浏览器中组装并生成 Excel 文件。

### 1. CSS 布局调整 (`query.css`)
需要更新以下样式以限制页面滚动：
*   `.query-dashboard-container` 设为 `height: 100vh; overflow: hidden;`。
*   `.content-area` 设为 `flex: 1; overflow: hidden;`。
*   `.report-panel` 设为 `flex: 1; display: flex; flex-direction: column; overflow: hidden;`。
*   `.table-container` 设为 `flex: 1; overflow: auto;`。

### 2. JS 导出处理逻辑
1.  **生成自然日序列**：
    从 `startDate` 递增到 `endDate` 生成所有的 `YYYY-MM-DD` 格式字符串数组。
2.  **提取有效考勤日期**：
    遍历所有学生的 records，找出所有至少含有一条考勤记录（无论 status 为 true 还是 false）的日期，形成 `activeDates` Set。
3.  **计算汇总数据**：
    *   `签到天数` = 记录中 status 为 true 的数量。
    *   `未签到天数` = `activeDates` 集合中该学生 status 不为 true（为 false 或无记录）的数量。
4.  **构建 AOA (Array of Arrays) 数据矩阵**：
    *   第一行：`['', ...students.map(s => s.name)]`
    *   第二行：`['签到天数', ...students.map(s => getPresentDays(s))]`
    *   第三行：`['未签到天数', ...students.map(s => getAbsentDays(s))]`
    *   第四行起（按自然日 `dateStr` 循环）：
        *   A列单元格：格式化为 `M月D日` 的字符串。
        *   若 `dateStr` 不在 `activeDates` 中，学生单元格全为空字符串。
        *   若 `dateStr` 在 `activeDates` 中，则依次填入 `✓ 备注` 或 `✗ 备注`。
5.  **生成并保存 Excel**：
    *   使用 `XLSX.utils.aoa_to_sheet` 将 AOA 矩阵转换为 Worksheet。
    *   设置单元格列宽以防止文字遮挡。
    *   使用 `XLSX.writeFile` 触发浏览器下载，文件命名为：`{className}_考勤报表_{startDate}_至_{endDate}.xlsx`。
